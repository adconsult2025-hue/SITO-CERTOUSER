Home · Metodo

“Prima si decide. Poi si investe.”

Un metodo in cinque fasi che rende chiare le assunzioni, tracciabili le decisioni e verificabili gli output.

Fasi, input e output

1. Inquadramento

Input: obiettivi, asset, vincoli normativi. Output: perimetro e requisiti di base.

2. Assunzioni e dati

Input: dati energetici e finanziari, ipotesi di prezzo. Output: simulazioni con note metodologiche.

3. Decisione

Input: verbali, deleghe, pareri tecnici. Output: delibera tracciata con responsabilità.

4. Attuazione

Input: documenti autorizzativi, contratti, check-list. Output: pratiche evase con log.

5. Monitoraggio

Input: dati di esercizio, KPI, baseline. Output: report e piani di miglioramento.

Deliverable

Report decisionali

Logiche, assunzioni e scelte esplicite.

Checklist

Adempimenti, permessi, scadenze.

Data room

Documenti indicizzati con versioni e permessi.

Regole di governance

Ruoli, deleghe e workflow autorizzativi.
